# Machine Vision & Advanced Image Processing

## 📝 Module Overview
This repository contains a comprehensive portfolio of vision algorithms developed during the **ACS6123: Machine Vision** module at the University of Sheffield. The module is 100% coursework and project-driven, focusing on the mathematical foundations and practical implementations of computer vision—moving from pixel-level manipulation to deep learning-based object detection.

## 🚀 Key Topics Covered
* **Image Foundations:** Pixel-point operations, multi-image processing, and color histogram analysis.
* **Feature Extraction:** Edge detection algorithms, texture analysis, and geometric transformations.
* **Segmentation & Motion:** Image segmentation techniques, optical flow methods, and motion detection.
* **Target Tracking:** Automatic object detection in video sequences using Bayesian tracking frameworks.
* **Deep Learning:** Classification and localization utilizing Convolutional Neural Networks (CNNs).

## 🛠️ Tech Stack & Toolboxes
* **Languages:** MATLAB / Python
* **Key Frameworks:** Image Processing Toolbox, Computer Vision Toolbox, Deep Learning Toolbox
